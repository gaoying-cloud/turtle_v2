## 两种再平衡模式对比

### daily — 每日权重复位

每日收盘后将权重强制拉回 50/50（净值层面加权: `combo = 0.5×N + 0.5×T`）。
等价于每天调仓至目标权重，会系统性地压低组合波动率。

| 指标 | 组合 |
|------|------|
| CAGR | 15.1% |
| 年化波动率 | 13.0% |
| Sharpe | 1.14 |
| 最大回撤 | -11.9% |
| Calmar | 1.27 |

### none — 无再平衡（权重自然漂移）

仅初始配置 50/50 后不再调仓（收益率层面加权: `r_combo = 0.5×r_n + 0.5×r_t`）。
策略权重随各自表现自然漂移，海龟表现优于 N字时实际组合中海龟权重会逐步提高。

| 指标 | 组合 |
|------|------|
| CAGR | 15.0% |
| 年化波动率 | 13.2% |
| Sharpe | 1.12 |
| 最大回撤 | -12.3% |
| Calmar | 1.22 |

### 模式差异分析

| 维度 | daily (每日再平衡) | none (无再平衡) | 差异 |
|------|-------------------|-----------------|------|
| 波动率 | 13.0% | 13.2% | daily 低 0.2pp |
| Sharpe | 1.14 | 1.12 | daily 高 0.02 |
| MDD | -11.9% | -12.3% | daily 浅 0.4pp |
| Calmar | 1.27 | 1.22 | daily 高 0.05 |
| CAGR | 15.1% | 15.0% | 几乎一致 |

**结论**: 两种模式核心指标差异很小（<0.5pp）。daily 模式因每日再平衡，波动率和回撤略微改善；none 模式在强势策略（海龟）跑赢时，组合自然向强者倾斜。当前因双策略 CAGR 差距较大（9.3% vs 19.6%），none 模式在 2025 年海龟暴涨时产生了显著漂移（35.5% vs 31.9%），但整体差异有限。

**推荐**: 实盘场景使用 **daily** 模式（每日再平衡），因为：① 风险控制更优（波动率/回撤更低）；② 权重始终可知，便于资金调度；③ 与后续风险平价/动态权重模式接口一致。

## 逐年收益对比

| 年份 | N字 | 海龟 | daily 组合 | none 组合 | daily−none |
|------|-----|------|-----------|----------|------------|
| 2020 | 15.9% | 32.0% | 23.9% | 24.0% | −0.1pp |
| 2021 | 1.1% | 4.6% | 3.0% | 4.0% | −1.0pp |
| 2022 | 4.6% | 10.8% | 8.0% | 9.1% | −1.1pp |
| 2023 | 12.5% | 32.7% | 23.7% | 22.4% | +1.3pp |
| 2024 | 3.5% | 0.9% | 1.9% | 2.6% | −0.7pp |
| 2025 | 16.3% | 48.9% | 35.5% | 31.9% | +3.6pp |
| 2026 | 5.0% | 0.5% | 2.1% | 2.9% | −0.8pp |

**逐年差异最大年份**: 2025 年 daily−none = +3.6pp。该年海龟暴涨 48.9%（远超 N字 16.3%），none 模式下海龟权重因不调仓而逐步增大（实际权重偏离 50/50），导致组合在海龟巅峰时承担了更大的海龟回撤风险（2026 年海龟仅 +0.5%，none 组合 +2.9% vs daily +2.1%，none 反而略优 — 因为海龟权重已被 2025 年推高）。

## 组合优势总结

| 检查项 | 结果 | 详情 |
|--------|------|------|
| 夏普提升 | ⚠️ 未达最优 | 1.14 (daily) vs 海龟单策略 1.27 |
| 回撤降低 | ✅ 显著改善 | −11.9% vs −18.5% (海龟) / −21.9% (N字) |
| 波动率降低 | ✅ 显著改善 | 13.0% vs 15.0% (海龟) / 19.1% (N字) |
| Calmar 提升 | ✅ 超越单一 | 1.27 (daily) vs 1.06 (海龟) / 0.42 (N字) |
| 收益接近最优 | ⚠️ 差距较大 | 15.1% vs 19.6% (海龟) — 仅达最优的 77% |

组合的核心价值在于**风险分散**：虽然 CAGR 未能超越海龟，但回撤从 −18.5% 降至 −11.9%（降幅 36%），波动率从 15.0% 降至 13.0%（降幅 13%），Calmar 比率从 1.06 提升至 1.27。适合风险厌恶型资金配置。
