## turtle_v2 完整总结

### 当前配置（V4.0）

6 只跨市场 ETF（中证500、中证1000、创业板、科创50、纳指ETF、黄金ETF），海龟突破入场（`close` 确认），三重退出保护（2N固定止损 + 移动止损只上移 + 10日反向突破），仓位公式 `/(2·N)`，α-风险平价权重，渐进式集中度熔断，滑动窗口累计亏损暂停。

回测区间 2020-01-01 ~ 2026-06-10，初始资金 ¥200,000，年化约 +1.1%，胜率约 41%，最大回撤约 7.7%。

### 关键结论

| # | 结论 | 证据 |
|:--|:--|:--|
| 1 | **止损必须优先更新再检查退出** | `_update_trailing_stop` 必须在 `_should_exit` 之前调用，否则三重保护形同虚设 |
| 2 | **入场用 `close`、止损用 `low` 是最优组合** | 实验证明不对称入场和止损使得假突破被过滤，真止损更敏捷，贡献约 +9pp |
| 3 | **仓位公式 `/(2·N)` 显著放大仓位** | 公式对齐前仓位约为对齐后的 1/3.6，对齐后收益明显改善 |
| 4 | **跨市场分散是不可替的核心** | 六只跨市场品种（含纳指和黄金）比纯 A 股组合高约 11pp 收益 |
| 5 | **震荡市改进均无效** | ADX、ATR扩张、时间止损、排除区间四个实验全部不如原版 |
| 6 | **海龟策略依赖趋势** | 2020-2026 是 A 股震荡市——唯一的利润来源是纳指和黄金的独立趋势 |

### 适用情形

| 环境 | 表现 |
|:--|:--|
| **强趋势市**（2019, 2024.9 之后的 A 股） | ✅ 最佳：趋势越长、越强，海龟越有效 |
| **宽幅震荡市**（2021-2023 的 A 股） | 🟡 一般：频繁止损割肉，利润来源是纳指和黄金的独立行情 |
| **长期阴跌**（2022 全年） | 🔴 不适合：需要减少交易频率或暂停 |
| **跨市场分散** | ✅ 关键：海龟需要低相关性的趋势来源才能生存 |

### 后续试验方向

不再寻找震荡市的过滤器（已证明均无效），改为主攻更根本的方向：

| 方向 | 内容 |
|:--|:--|
| **1. 品种扩展** | 引入更多与 A 股低相关的品种（如豆粕、标普 500、白银）→ 扩大趋势来源 |
| **2. 参数自适应** | 根据当前 ATR(20)/ATR(60) 动态调整 `risk_per_unit`——降低震荡市仓位，加大趋势市仓位，而非禁止交易 |
| **3. 多时间框架** | 从日线扩展到周线维度——周线趋势过滤日线入场，过滤假信号时有更少的延迟 |
| **4. 空头实现** | 完整实现做空逻辑——A 股虽然没有融券，但黄金和纳指ETF 支持 T+0 日间做空（需回测确认） |
| **5. S9 Dry-Run 准备** | 将 V4.0 策略通过信号校验脚本转入模拟盘验证 |

其中 **#2 参数自适应** 是目前最有效且改动量最小的方向。
