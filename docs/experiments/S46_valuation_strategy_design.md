## 一、策略起源

924 报告的核心发现：
- 924 前创业板 PE=23.80（近十年 **0.66%** 分位），PB=3.04（<5% 分位）——历史级极端低估
- 仅 6 个交易日 PE 暴涨 +63.2%（23.80→38.84），随后 3 个月均值回归至 30.74
- **纯估值脉冲，盈利基本面全程缺席**
- 尖峰 PE 38.84 仅处于 ~28% 分位，仍低于中位数 ~42 倍，未进入泡沫区间
- PB-ROE 框架：924 前 PB 处于 10% 分位而 ROE 处于 94% 分位——极端错配构成上涨初始合理性

**策略本质**：基于 PE/PB 历史分位的**价值回归策略**——极端低估时分批建仓，极端高估时分批减仓。

**与海龟策略的关系**：本质不同，独立运行。
- 海龟策略 = 趋势跟随（追涨），信号来自价格突破
- 估值策略 = 价值回归（抄底），信号来自估值分位 + 价格确认
- 两者信号在时间轴上天然错位，**不做融合，各管各的资金池**

## 三、核心信号设计

### 3.1 PE 分位计算

| 参数 | 设定 | 理由 |
|:---|:---|:---|
| 分位窗口 | 5 年 + 10 年混合 | 兼顾历史规律与近期市场结构 |
| 混合方式 | `max(pct_5y, pct_10y)`（取保守值） | 建仓信号宁缺毋滥，双窗口确认 |
| 阈值调整 | "极端低估"阈值放宽至 <10% | 适配保守分位的系统性偏高 |

### 3.2 入场规则：PE 分位 + 价格行为双重确认

```
核心原则：PE 分位管"值不值得买"，价格行为管"现在合不合适"
```

**时间 × 空间建仓矩阵**：

```
                    价格还在跌          价格横盘           价格开始涨
                    (距60日低点<3%)    (窄幅震荡)        (站上MA20)
──────────────────────────────────────────────────────────────────
PE < 5%             观察，不出手         第1批(30%)        第2批(30%)
PE 5-10%            观察，不出手         不出手            第1批(20%)
PE 10-15%           不出手              不出手            第1批(10%,试探)
PE 15-50%           不出手              不出手            不出手（持有）
PE 50-70%           —                   —                第1次减仓
PE > 70%            —                   —                第2次减仓→清仓
```

**入场条件（全部满足）**：
1. PE_TTM 分位 < 10%（双窗口混合取保守值）
2. PB_LF 分位 < 15%
3. EPS 隐含值环比（3 个月）> -3%（盈利未显著恶化）
4. 价格确认信号（距 60 日低点反弹 > 3% 或站上 MA20）

### 3.3 出场规则

| 条件 | 动作 |
|:---|:---|
| PE_TTM 分位 > 70% | 减仓 50% |
| PE_TTM 分位 > 85% | 清仓 |
| PE 周涨幅 > 15% 且 EPS 周涨幅 < 2%（被动拔估值） | 紧急减仓至 ≤30% |
| EPS 隐含值环比（3 个月）< -10%（盈利恶化） | 减仓 50% |

### 3.4 PE 空间锚

```
创业板 PE 历史区间：23.10 ~ 74+（10年），中位数 ~42

PE 区间         动作                价格隐含空间
─────────────────────────────────────────────
23-25（<5%）    第1批 + 第2批       潜在 +80%（至中位数）
25-30（5-15%）  第3批               潜在 +40-60%
30-42（15-50%） 持有不动             等待均值回归
42-55（50-70%） 第1次减仓 50%       已到中位数以上
55+（>70%）     第2次减仓 → 清仓    接近泡沫区
```

## 四、资金管理

### 4.1 组合层面资金分配

```
总资金 100%：
├─ 海龟策略：50%    趋势跟随
├─ N字结构：20%     形态识别
├─ 估值策略：20%    价值回归  ← 新增
└─ 现金/国债：10%   流动性储备
```

估值策略只占 20%，原因：
- 交易频率最低（几年一次完整建仓→减仓周期）
- "0.66% 分位"级别的机会不是每年都有

### 4.2 策略层面资金管理

- 分批建仓，最多 3 批
- 每批占总策略资金的 30-33%
- 保留 10% 现金应对极端情况

## 五、风控机制

### 5.1 价值陷阱防护

```python
def compute_eps_penalty(price_series, pe_series, lookback_days=63):
    """
    EPS 隐含趋势检测（替代完整 PB-ROE 错配）
    lookback_days=63 ≈ 3 个月交易日
    """
    eps = price_series / pe_series  # 隐含 EPS
    eps_change = eps.iloc[-1] / eps.iloc[-lookback_days] - 1

    if eps_change < -0.10:    # 季度恶化超 10%
        return 0.5            # 强制减半
    elif eps_change < -0.03:  # 轻微恶化
        return 0.8
    else:
        return 1.0
```

### 5.2 极端行情紧急响应

```
if PE周涨幅 > 15% AND EPS周涨幅 < 2%:
    → 被动拔估值，紧急减仓至 min(current_cap, 0.3)
```

### 5.3 时间止损

| 条件 | 动作 |
|:---|:---|
| PE <5% 持续 >12 个月无反转 | 检讨是否价值陷阱，降至半仓 |
| 第 1 批建仓后浮亏 >20% | 暂停加仓，等待价格企稳 |

## 六、与海龟策略的关系

### 6.1 关键矛盾

```
海龟：价格涨了 → 买入（追涨），PE 高估时往往是海龟进场时间
估值：便宜了   → 买入（抄底），PE 低估时海龟往往还在观望
```

**不做融合**。之前尝试"估值模块调节海龟参数"的方案已废弃。

### 6.2 独立运行

```
┌─────────────────────────────────────────────┐
│                  组合配置层                    │
│    资金分配：海龟50% / N字20% / 估值20%       │
├──────────────┬──────────────┬────────────────┤
│  海龟策略     │  N字结构策略   │  估值策略(NEW)  │
│  趋势跟随     │  形态识别      │  价值回归      │
│  自有信号     │  自有信号      │  自有信号      │
│  独立资金池   │  独立资金池    │  独立资金池    │
└──────────────┴──────────────┴────────────────┘
```

唯一共同约束：组合层面的总资金分配比例。

### 6.3 为什么不做融合

1. 信号时间轴天然错位——强行融合 = 互相掐架
2. 924 报告显示：PE 28% 分位（10/8 尖峰）事后看仍是偏低，此时估值不该阻断海龟
3. 磨损是趋势策略的刚性成本，不应被估值信号消灭，只能被管理

## 十、参考文献与相关研究

> 以下为 2026-07-21 针对 PE/PB/ROE 估值策略的专项文献调研，按与本文策略的相关度排列。

### 10.1 A 股直接实证：PE/PB 分位控制仓位法

**雪球量化研究（2013-2026）**：沪深 300 指数上用 PE/PB 5 年滚动分位做每日仓位调节。

| 策略 | 年化收益 | 平均仓位 |
|:---|:---|:---|
| 买入持有（基准） | 5.05% | 100% |
| **PE 5年分位** 控制仓位 | **6.40%** | 67.3% |
| **PB 5年分位** 控制仓位 | **7.00%** | 77.3% |

关键发现：
- **PB 分位优于 PE 分位**——PB 更稳定，平均仓位更高（77.3% vs 67.3%）
- 超额收益来自"熊市少亏 > 牛市少赚"（A 股牛短熊长特征）
- 公式：`仓位 = 100% - (当期估值 - 最低估值) / (最高估值 - 最低估值)`

> → **对本设计的启示**：考虑提高 PB 在信号中的话语权。当前设计中 PB 仅作为辅助确认（PB 分位 < 15%），可将其提升为与 PE 等权的独立信号。
>
> **来源**：雪球 @持有封基 / 网易

### 10.2 学术基准：Shiller CAPE（周期调整市盈率）

Nobel 奖得主 Robert Shiller 提出的 CAPE = Price / 10 年平均通胀调整后盈利，是 PE 分位策略的学术起源。

**Barclays Shiller CAPE US 行业轮动（2002-2012）**：

| 指标 | CAPE 轮动 | S&P 500 TR |
|:---|:---|:---|
| 10 年化收益 | **10.8%** | 7.2% |
| 超额 | +3.6%/年 | — |
| 年化波动率 | 19.6% | 21.2% |
| 夏普 | 0.74 | 0.54 |
| 最大回撤 | -43.3% | -55.3% |

规则：选最便宜的 4 个 GICS 行业 + 剔除 12 个月动量最差的 1 个，月度再平衡。

**关键验证**：活交易 10 年（2012-2022）与回测一致，alpha 来自相对 CAPE 信号本身而非运气（Dumontier & De Franco, 2023）。

> → **对本设计的启示**：**估值（Value）+ 动量（Momentum）的组合**经过学术和业界双重验证。我们的"PE 分位 + 价格行为双重确认"本质是同一逻辑——估值为锚选时机，动量过滤防价值陷阱。
>
> **来源**：Barclays QIS Insights / Shiller (2000) *Irrational Exuberance* / Dumontier & De Franco (2023)

### 10.3 全球 32 国 CAPE 分位策略（2008-2024）

QuantConnect 平台上用 CAPE < 15（绝对阈值）+ 最便宜 33%（分位）双过滤，横跨发达国家 + 新兴市场，年度再平衡，2x 杠杆。

| 指标 | 数值 |
|:---|:---|
| 年化收益 | 5.06% |
| 最大回撤 | 29.7% |
| 胜率 | 89% |
| Beta | 0.247 |

> → **对本设计的启示**：**绝对 PE 阈值 + 分位双过滤**比纯分位更稳健。当前设计只用了分位（PE < 10% 分位），V2 可考虑加入绝对 PE 阈值（如 PE_TTM < 25 AND 分位 < 10%）。
>
> **来源**：paperswithbacktest/awesome-systematic-trading

### 10.4 华泰证券：PB-ROE 残差因子策略（2011-2021）

基于 Jarrod Wilcox（1984）PB-ROE 定价模型，用回归残差构建估值因子。在中证 800 成分股回测。

核心结论：
- PB-ROE 残差因子有效性**接近 PE，强于 PB 和 PS**
- ROE 尾部剔除（去掉最差 20%）后效果显著提升
- 残差因子构建的价值动量组合，长期表现优于纯 PE 因子策略，2018 年后优势尤为明显

> → **对本设计的启示**：PB-ROE 错配检测（残差法）是 V2 的明确升级方向。当前 V1 的 EPS 隐含趋势是简化版代理。
>
> **来源**：华泰证券金工团队（2021）《PB-ROE 策略框架系列之一：估值指标再审视》

### 10.5 国投证券：综合估值因子选股（2024）

BP/EP/CFP 三因子融合 + 估值中枢稳定性因子 + ROE 调整，在中证 800 回测（2010 年起）。

| 指标 | 数值 |
|:---|:---|
| 年化收益 | **25.64%** |
| 相对中证 800 超额 | 25.41%/年 |
| 信息比率 | **2.62** |
| 相对偏股基金超额 | 20.97%/年 |

> ⚠️ 这是**选股层面**（在中证 800 中挑股票），非指数层面。选股的超额空间远大于指数择时。我们的策略是指数级，预期超额约 1-3%/年。
>
> **来源**：国投证券（2024）《PB-ROE 框架选股策略》

### 10.6 招商证券：四维择时框架 2.0

用 **ERP（股权风险溢价）** 替代 PE，窗口 12 年，阈值收紧至 (2.5%, 97.5%)。万得全 A 回测（2004 年起）。

| 指标 | 策略 | 基准 |
|:---|:---|:---|
| 年化 | **22.23%** | 10.83% |
| 最大回撤 | **24.95%** | 70.59% |
| 夏普 | **1.09** | 0.45 |

> → **对本设计的启示**：ERP 可能比纯 PE 分位更好——`ERP = 1/PE - 10年期国债收益率`，同时包含估值和利率环境。V2 升级方向。
>
> **来源**：招商证券（2025）《权益择时系列之二：四维择时框架 2.0》

### 10.7 民生证券：横纵向二维估值法

将估值拆为横向（同业分位）+ 纵向（自身历史分位），含 PB-ROE 和财务指标调整。

结论：
- 横向估值越低，抗回撤能力越强（低估值组最大回撤比无效组低约 15pp）
- 纵向估值低的标的在上涨行情中弹性更强（估值分位 >80% 的标的涨幅显著落后）

> → **对本设计的启示**：印证了我们的核心假设——**估值策略的真正价值不在超额收益，而在降低最大回撤。**
>
> **来源**：民生证券（2025）《可转债策略系列：横/纵向估值法挖掘正股估值性价比》

### 10.8 动态 DCA 策略（2025 学术论文）

用归一化 CAPE 风险指标（0-1 连续值）驱动定投金额——CAPE 低时多投、CAPE 高时少投或减仓。回测显示优于传统固定金额定投。

> → **对本设计的启示**：与我们的设计思路完全一致——PE 分位连续映射到仓位权重，而非二元买入/卖出。
>
> **来源**：UPC Barcelona (2025) "Dynamic DCA Using Normalized CAPE Risk Metric"

| PB 分位在 A 股优于 PE 分位 | 提高 PB 话语权，考虑 PE/PB 等权或 PB 主导向 |
| Value + Momentum 是验证过的最佳组合 | 坚定 PE 分位 + 价格行为双重确认的设计 |
| 绝对阈值 + 分位双过滤更稳健 | V2 加入 PE_TTM < 25 绝对条件 |
| 指数级策略超额 ~1-3%/年 | 管理预期——策略价值在降回撤，非追求高收益 |
| ERP 替代 PE 可能更好 | V2 升级方向（含利率环境信息） |
| PB-ROE 残差框架是进阶路线 | V2 升级方向（与当前 V1 EPS 代理互补） |

## 十一、讨论记录

### 2026-07-21：设计讨论

**参与**：用户 + Claude

**关键决策节点**：

1. **方案 C（全市场仓位总控）→ 独立策略**：最初设想用估值模块调节海龟仓位，经过三轮讨论后发现两者信号本质矛盾（动量 vs 价值），最终决定做独立策略。

2. **融合尝试 → 放弃**：尝试了"估值模块调节海龟入场/止损/仓位参数"的设计，但用户指出"PE/PB 高估时往往是海龟进场的时间"——这是矛盾的根因。

3. **仓位映射的迭代**：
   - 第一版：线性递减（30-60% 分位 → 40-20% 仓位），过于压制海龟
   - 第二版：宽中枢（30-70% 分位固定 0.85），但仍试图与海龟融合
   - 最终版：独立策略，PE 分位 + 价格行为双重确认建仓

4. **混合分位方式**：5y+10y 取 max()（保守值），阈值从 <5% 放宽至 <10%。

5. **EPS 隐含趋势**：3 个月环比窗口，< -10% 减半仓。

6. **事件触发**：PE 周涨 >15% + EPS 周涨 <2% → 被动拔估值紧急减仓。

7. **时间 × 空间建仓矩阵**：PE 分位管"值不值得买"，价格行为（距 60 日低点反弹/站上 MA20）管"现在合不合适"。
