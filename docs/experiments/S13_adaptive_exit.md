# 实验: 自适应退出机制

## 元数据
- 提出: 2026-07-08
- 分支: —
- 状态: ❌ 不推荐（见结果）

## 假设
趋势向上时宽容出场（stop_period=10），非趋势时严格出场（stop_period=6），
可改善回撤控制同时保留利润奔跑空间。

## 成功标准
- 组合 Sharpe ≥ 当前 0.81
- MDD 从 18.45% 降至 ≤ 16%

## 结果

### A/B 测试（2026-07-08）
`regime_filter`（入口拦截）测试结果：
- CAGR: 14.05% → 8.32%（-5.74pp）
- 夏普: 1.04 → 0.60
- 终值: ¥517k → ¥271k

### 结论
❌ **当前优先级不高**。
1. 当前 `stop_period=8` 的 OOS Sharpe=1.04 已很好
2. 入口拦截测试显著劣化，出口切换虽未测试但增量空间有限
3. 建议先调优 `MarketRegime` 子指标再考虑
