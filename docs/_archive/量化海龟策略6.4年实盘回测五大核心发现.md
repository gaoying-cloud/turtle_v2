# 量化海龟策略6.4年实盘回测五大核心发现
## 项目基础说明
本项目基于**6.4年实盘级回测数据**，完成5大量化交易核心命题验证，核心结论：策略骨架（低相关组合体系）远优于单一交易策略本身。

## 一、5大核心发现
### 1. 分散化 > 单一品种择时能力
低相关性对冲带来的组合增益，远超挑选强势品种的收益。

| 品种代码 | 单兵独立收益 | 组合贡献收益 | 核心结论 |
| ---- | ---- | ---- | ---- |
| 黄金ETF 518880 | -4% | +728,563 | 组合额外增收73万，自身单兵亏损，依靠和股票低对冲抵消回撤 |
| 中证500 510500 | +28% | +319,863 | 组合进一步放大原有盈利 |

核心逻辑：黄金单独持仓亏损，但与股指走势低相关，对冲平滑账户波动，成为组合第一利润来源；并非品种变强，而是跨品种分散分担单边行情风险。

### 2. 信号过滤器是双刃剑
单品种运行时过滤器易形成负反馈死循环：
> 单笔亏损 → 过滤跳过3个后续信号 → 行情反转强制入场 → 再次亏损 → 持续过滤信号

多品种组合模式下该缺陷被抵消：黄金信号被过滤空仓阶段，中证500往往产生正向收益，账户净值不会单边持续回撤，机制风险可控。

### 3. 劣质亏损品种长期持有会大幅侵蚀收益
回测中错误保留2只持续亏损品种长达2年，数据如下：

| 品种代码 | 交易笔数 | 累计总亏损 | 亏损占初始本金比例 |
| ---- | ---- | ---- | ---- |
| 科创50 588000 | 3笔全部亏损 | -191,792 | -43% |
| 中证1000 159845 | 1笔亏损 | -35,224 | -8% |

剔除两只亏损品种前后对比：
- 整体总收益：+122% → +221%
- 最大回撤：46.86% → 38.29%

结论：品种数量并非越多越好，**低相关性、正向收益品种**才是组合核心，劣质标的必须及时剔除。

### 4. 平均资金利用率仅38%，属于策略固有设计特征
海龟策略定位：高盈亏比、低胜率、低频趋势跟踪策略
- 年均交易次数：9次，月均仅0.8次
- 全年62%时间账户持有现金，平均资金利用率38%
- 整体盈亏比1.97，最终实现总收益+210%

解读：低资金占用不是缺陷，是策略设计逻辑；海龟策略核心目标捕捉中长期大级别趋势，大部分震荡行情主动空仓规避亏损。

### 5. 初始本金下调，收益率几乎无衰减
ATR动态仓位公式跟随账户净值自动缩放仓位，仅改变盈利亏损绝对金额，不影响收益比例。

| 指标 | 初始资金20万 | 初始资金12万 | 变动说明 |
| ---- | ---- | ---- | ---- |
| 总收益 | +221% | +210% | 收益水平基本持平 |
| 复合年化CAGR | 19.86% | 19.24% | 仅下滑0.6%，影响极小 |
| 平均资金利用率 | 25% | 42% | 本金降低后资金使用效率显著提升 |

## 二、项目整体一句话总结
本项目验证核心并非“海龟趋势策略能够盈利”，而是**低相关资产组合+劣质品种剔除+参数体系优化**三层风控架构，是量化稳定盈利的核心，优先级高于单一交易信号策略。

## 三、通用可复用体系框架
交易信号代码可自由替换（双均线、动量突破、布林带等），底层通用骨架永久复用：
> 4只跨市场ETF配置 + 品种间低相关持仓分散结构
