# 实验管理（缩写）

是 `/experiment` 的别名。用法相同。

- `start <name>` — 开新实验
- `run` — 跑回测
- `check` — 检查结果
- `pass` — 通过合并
- `fail` — 失败关闭
- `status` — 当前状态
- `list` — 实验清单
